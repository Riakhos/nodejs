API Express

Cette API Express permet de gérer des utilisateurs et des messages avec Node.js, Express et MongoDB.
Fonctionnalités :

  • Authentification sécurisée (JWT)
  • Gestion des utilisateurs (CRUD)
  • Gestion des messages (CRUD)
  • Recherche de messages par utilisateur ou par identifiant
  • Interface HTML pour tester toutes les fonctionnalités
Démarrez le serveur puis utilisez l'interface web ou un outil comme Postman pour tester les endpoints.

Comment démarrer ?

  • Installer les dépendances : npm install
  • Lancer le serveur : npm start
  • Accéder à l'interface sur localhost:8000

Exemples d'endpoints

  • POST /api/user/add : Inscription d'un utilisateur
  • POST /api/user/login : Connexion d'un utilisateur
  • GET /api/user/all : Liste des utilisateurs
  • POST /api/message/add : Ajouter un message
  • GET /api/message/all : Liste des messages
  • PUT /api/message/update/:idMessage : Modifier un message
  • DELETE /api/message/delete/:idMessage : Supprimer un message
