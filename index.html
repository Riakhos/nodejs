Projets Node.js

Bienvenue sur la page d'accueil des projets Node.js de Richard Bonnegent

  • API Express

    API REST sécurisée avec Express, MongoDB et JWT.
    Gestion complète des utilisateurs et des messages, authentification par JWT, recherche, pagination, interface HTML de test.
    Idéal pour découvrir la création d’API modernes avec Node.js.

    • CRUD utilisateurs et messages
    • Recherche par utilisateur ou message
    • Connexion MongoDB locale ou Atlas
    • Endpoints sécurisés
    • Interface de test intégrée
    Voir la documentation complète

  • Front React

    Interface utilisateur moderne en React pour consommer l’API Express.
    Affichage dynamique, design responsive, gestion des états, formulaires connectés à l’API.
    Parfait pour apprendre à connecter un front React à une API Node.js.

    • Affichage et recherche des utilisateurs/messages
    • Ajout, modification, suppression en temps réel
    • Optimisations (voir OPTIMISER.md)
    Voir la documentation complète
